Installing, Uninstalling, and System Requirements

System Requirements

A colour display screen with a resolution of at least 800 * 600
Windows 10, with a sound card.
Transcribe! is also available for Mac and Linux.

Installing Transcribe!

If you are reading this then you probably already have installed it. But for completeness : Transcribe! for Windows is distributed as a self-installing executable (built with the very wonderful Inno Setup). Run the file (double click it) to install Transcribe!, and follow the instructions which appear.

If you want to back up Transcribe! then you should save the downloaded installation file. Other than that, there is no need to keep the downloaded file once Transcribe! is installed.

If You Want To Uninstall Transcribe!

"Start Menu" - "Settings" (the Gear icon) - "Apps". You can also use Control Panel if you prefer.
