Misc Effects Page
This page has a simple volume control at the right, which is always active and only affects the current view, which is why this page does not have the usual "Active/Bypass" and "Apply to all views of this transcription" controls.
This control works by scaling the sample values, which is a quite separate matter from your computer's volume control which will also work. If you turn it up high you will get digital overload - you should instead turn up your computer's volume control if you want to increase the overall level. Transcribe!'s volume control is useful for bringing up the level of a sound file which was originally recorded too quiet. Transcribe! has a compressor/limiter so if you do turn the volume too high you will get compression rather than distortion, and a red bar will appear below the volume slider in the lower toolbar to show you the amount of compression.

There are in fact several other ways of adjusting the volume too. There is a slider in the lower toolbar. There are keyboard shortcuts, and you can use a MIDI continuous controller.

At the far right is the "Clicks" slider. If you set this to any value greater than zero then it will add metronome clicks during playback, wherever there is a marker. The sound used was recorded from a genuine old wooden metronome. To use it you will probably need to add beat markers if you haven't already, but remember that you can often do this most easily using automatic subdivisions (see the Help for the Markers menu). If you are getting an uneven beat then remember that there is an "Equally space markers" command on the Markers menu. Also be aware that if the music was originally played organically (not to a click track) then there can be subtle variations of timing between one measure and another. This is not an error or inaccuracy, it's called music! However if the music is played at an exactly constant tempo then you should be able to get accurate placement of markers with the "Equally space markers" command.
It can happen that when you tap the keyboard shortcuts to add markers while the music is playing, the markers end up late. In that case please see the Latency setting here.

At the upper left is the "At" box which reports the current position (the red triangle), and the "Selected" box which reports the width of the current selection, if any. These are in minutes and seconds and refer to timings in the original sound file, ignoring any speed alteration you may have made. You can type new numbers here if you want. Use either 'm' or a colon to separate the minutes from the seconds. You can't type a space because this will be interpreted as a shortcut - however you don't need spaces in these boxes. These numbers also appear in the lower toolbar.

Loops

Here you can store and recall any number of positions/selections, possibly for use as loops though they don't have to be. If you set a position and/or selection using the mouse on the waveform profile in the usual way and then shift-click one of the loop buttons then the position/selection will be remembered there and can be recalled with a click whenever you want. Remember that you will only get looping playback if, on the Play menu, "Loop" is on.

If you want to then you can also store Fx settings (speed, EQ etc) along with any loop, and those Fx settings will be restored when you select that loop. This means you can, for instance, store not only the loop position but also the speed or volume at which it is to be played.

Each loop button displays its number at the left, then it will say "Fx" if this loop has Fx settings stored with it, then there is space for a name which you can give it, and then a "Location indicator", giving an approximate graphical indication of where this loop is. Section markers are shown, along with the position of the stored loop. The full width of the Location indicator represents the full length of the track.

You can have as many loops as you want, and you will always have at least 20. Only 10 are displayed at a time, and you can scroll them with the scrollbar at the right. It can be useful to configure Recall commands as keyboard shortcuts if you use them a lot. Commands from LoopStore1 and LoopRecall1 up to LoopStore20 and LoopRecall20 are available, which access the correspondingly numbered loops. Loop numbers higher than 20 can only be accessed by pressing their buttons, not by keyboard shortcuts.

If you right-click a loop button then you get a pop-up menu allowing you to do various things:

  • Store: stores the current position/selection in this loop (same as shift-click).
  • Clear: clears the content of this loop, leaving it empty.
  • Delete: removes this loop altogether, moving subsequent loops up by one.
  • Insert Before or After: these insert a new empty loop, moving subsequent loops down by one.
  • Name: this allows you to give the loop a name, or edit its name if it already has one.
  • Color: this allows you to color the loop button, to help you remember what different loops might mean, if you have lots.
  • Store Fx: stores the current Fx settings in this loop.
  • Recall Fx: recalls the Fx settings from this loop (if there are any), without changing your current position or selection.
  • Clear Fx: clears the Fx settings from this loop (if there are any).

You can reorder loops by clicking and dragging any loop to a new location.

If you choose to "Show Loop Line" on the View menu then you will see these same loops displayed on the Loop Line in the main window, and you can create and select loops there instead.

Move

These buttons move your current position/selection to the left or right by various amounts. For this purpose, beat markers and automatically generated subdivision markers are considered equivalent. If you hold down the <shift> key then the button will increase the selection at the relevant end. And if you hold down both <shift> and <control> keys then the button will decrease the selection at the relevant end.
There are many more commands of this nature available for configuring as keyboard shortcuts if you want to.
