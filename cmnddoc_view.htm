Commands: View

Commands that can be used as Keyboard, Pedal, or Midi shortcuts as well as automation.

ViewCloseCanSaveCanVeto: If the current transcription has more than one view (because you have "Split" it) then this will "Unsplit". Otherwise it will close the current transcription leaving its window open but empty, giving the option to Save first, or Veto.
This is equivalent to the cross at the left hand end of the toolbar.

ViewSplit: Creates a new view of the current transcription.

ViewUnsplit: If you have split views then this removes the current view.

SynchronizeViewsToggle: If you have split views then this will make them scroll together, not independently.

ShowAllSoundTracks: If the sound file has sound tracks which are not currently displayed in any View then this will add more Views in order to show them all.

ViewSplitActivateAbove: If you have split views then this activates the one above the current one.

ViewSplitActivateBelow: If you have split views then this activates the one below the current one.

ViewSpectrumOn: Show Spectrum

ViewSpectrumOff: Hide Spectrum

ViewSpectrumToggle: Show/Hide Spectrum

ViewGuessNotesOn: Show note guesses

ViewGuessNotesOff: Hide note guesses

ViewGuessNotesToggle: Show/Hide note guesses

ViewGuessChordsOn: Show chord guesses

ViewGuessChordsOff: Hide chord guesses

ViewGuessChordsToggle: Show/Hide chord guesses

ViewCopyChordGuessToClipboard1: Copy the first chord guess (if there is one) to the clipboard. This will only work if chord guessing is turned on.

ViewCopyChordGuessToClipboard2: Copy the second chord guess (if there is one) to the clipboard. This will only work if chord guessing is turned on.

ViewCopyChordGuessToClipboard3: Copy the third chord guess (if there is one) to the clipboard. This will only work if chord guessing is turned on.

ViewCopyChordGuessToClipboardAll: Copy all chord guesses (if any) to the clipboard. This will only work if chord guessing is turned on.

ViewCopyChordGuessToTextBlock1: Copy the first chord guess (if there is one) to a new text block. This will only work if chord guessing is turned on and the text zone is shown.

ViewCopyChordGuessToTextBlock2: Copy the second chord guess (if there is one) to a new text block. This will only work if chord guessing is turned on and the text zone is shown.

ViewCopyChordGuessToTextBlock3: Copy the third chord guess (if there is one) to a new text block. This will only work if chord guessing is turned on and the text zone is shown.

ViewCopyChordGuessToTextBlockAll: Copy all chord guesses (if any) to a new text block. This will only work if chord guessing is turned on and the text zone is shown.

ViewDBOn: Show dB lines

ViewDBOff: Hide dB lines

ViewDBToggle: Show/Hide dB lines

ViewTimeLineOn: Show Time Line

ViewTimeLineOff: Hide Time Line

ViewTimeLineToggle: Show/Hide Time Line

ViewLoopLineOn: Show Loop Line

ViewLoopLineOff: Hide Loop Line

ViewLoopLineToggle: Show/Hide Loop Line

ViewVideoOn: Show Video

ViewVideoOff: Hide Video

ViewVideoToggle: Show/Hide Video

ViewSubtitlesToggle: Show/Hide Subtitles

ViewSubtitlesPrev: Previous Subtitle Source

ViewSubtitlesNext: Next Subtitle Source

ViewVideoFullScreenToggle: Full Screen Video

ViewFitSelection: Set the horizontal zoom so the present selection fits the window.

ViewFitWholeFileOn: Set the horizontal zoom so the whole file fits the window.

ViewFitWholeFileOff: Restore normal horizontal zoom. (Cancel "Fit whole file" mode).

ViewFitWholeFileToggle: Set the horizontal zoom so the whole file fits the window, or cancel this mode.

ViewSmallHorizZoomIn: Small horizontal zoom in.

ViewSmallHorizZoomOut: Small horizontal zoom out.

ViewBigHorizZoomIn: Big horizontal zoom in.

ViewBigHorizZoomOut: Big horizontal zoom out.

ViewVertZoomIn: Vertical zoom in

ViewVertZoomOut: Vertical zoom out

PianoRollShow: Show Piano Roll

PianoRollHide: Hide Piano Roll

PianoRollToggle: Show/Hide Piano Roll

PianoRollPianoOn: Include piano at left of piano roll.

PianoRollPianoOff: Do not include piano at left of piano roll.

PianoRollPianoToggle: Toggle inclusion of piano at left of piano roll.

PianoRollIncreaseSensitivity: Increase the sensitivity of the piano roll note detection, so more notes will be displayed.

PianoRollDecreaseSensitivity: Decrease the sensitivity of the piano roll note detection, so fewer notes will be displayed.

Commands that require parameters, which can only be used for Automation.

None.
