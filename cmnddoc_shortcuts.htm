Commands: Shortcuts

Commands that can be used as Keyboard, Pedal, or Midi shortcuts as well as automation.

MidiShortcuts: View and configure Transcribe!'s response to MIDI messages.

KeyboardShortcuts: View and configure keyboard shortcuts.

FootPedals: View and configure Transcribe!'s response to USB (not MIDI) foot pedals.

Commands that require parameters, which can only be used for Automation.

None.
